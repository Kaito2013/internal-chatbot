Internal Chatbot Widget

A lightweight, self-contained chat widget for embedding on any website.

Features

  • Self-contained Vanilla JS (no dependencies)
  • Session management with localStorage
  • Dark theme optimized for modern websites
  • Auth token support via window.chatbotConfig
  • Responsive design with mobile support
  • Public API for external control
  • Accessible (ARIA labels, keyboard support)

Quick Start

1. Basic Integration

<!-- Add this to your HTML -->
<script src="widget.js" 
  data-api-url="https://api.yourchatbot.com"
  data-position="right">
</script>

2. With Configuration

// Set config before widget loads
window.chatbotConfig = {
  authToken: 'your-jwt-token',  // Optional
  email: 'user@example.com',     // Optional
  offset: {                         // Optional
    bottom: 20,
    right: 20
  }
};

Public API

Control the widget from your code:

// Open the chat widget
window.Chatbot.open();

// Close the chat widget
window.Chatbot.close();

// Toggle open/close state
window.Chatbot.toggle();

// Clear chat history
window.Chatbot.clearHistory();

// Send a message programmatically
window.Chatbot.sendMessage('Hello!');

// Get current session
const session = window.Chatbot.getSession();

Event Listeners

// Listen for messages
window.Chatbot.on('message', (data) {
  console.log('New message:', data);
});

// Listen for session creation
window.Chatbot.on('sessionCreate', (session) {
  console.log('Session created:', session.id);
});

API Response Format

Your API should return responses in this format:

{
  "answer": "The response text from the chatbot",
  "sources": ["source1", "source2"],
  "used_crm": true
}

Custom Styling

Override CSS variables before the widget loads:

/* Change accent color */
.chatbot-widget {
  --cb-accent: #10b981;        /* Green */
  --cb-accent-hover: #059669;
}
